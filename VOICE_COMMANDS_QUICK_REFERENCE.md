# Voice Commands Quick Reference

## 🎤 Quick Start
1. Tap the microphone button (🎤) on any page
2. Wait for "Listening..." message
3. Speak your command clearly
4. Listen for voice confirmation

---

## 💡 Lights

### All Lights
```
"Turn on all lights"
"Turn off all lights"
"All lights on"
"All lights off"
```

### By Name (Replace [name] with your device name)
```
"Turn on [ceiling light]"
"Turn off [table lamp]"
"Switch on [bedroom light]"
"Switch off [hall light]"
```

### By Number
```
"Turn on light 1"
"Turn off light 2"
```

---

## 🌬️ Fans

### Basic Control
```
"Turn on [bedroom fan]"
"Turn off [living room fan]"
"Turn on fan 1"
"Turn off fan 2"
```

### Speed Control
```
"Set [bedroom fan] to 50 percent"
"Set fan 1 to 75 percent"
"Set [ceiling fan] to low speed"
"Set [bedroom fan] to medium speed"
"Set fan to high speed"
"Fan maximum"
```

Speed Presets:
- **Low**: 25%
- **Medium**: 50%
- **High**: 100%

## 💡 Pro Tips

1. **Use Your Device Names**: Instead of "light 1", say "ceiling light" or whatever name you gave it
2. **Be Natural**: Multiple variations work - "turn on", "switch on", "enable" all work
3. **Listen for Feedback**: The app will speak back confirming your command
4. **Check Device Names**: Go to Control page and click Edit icon to see/change device names
5. **Speak Clearly**: Wait for "Listening..." before speaking

## 🎯 Example Conversations

**Morning Routine:**
```
You: "Turn on all lights"
App: "Turning on all lights"

You: "Set bedroom fan to high speed"
App: "Setting bedroom fan to high speed"

You: "What is the temperature?"
App: "The current temperature is 24 degrees celsius"
```

**Adjusting a Specific Device:**
```
You: "Turn off table lamp"
App: "Turning off table lamp"

You: "Set living room fan to 50 percent"
App: "Setting living room fan to 50 percent"
```

**Checking Status:**
```
You: "What is the power consumption?"
App: "The current power consumption is 150 watts"

You: "Is there any motion?"
App: "Motion status: No motion detected"
```

## ✅ Works With

- Device names (customizable in Control page)
- Device numbers (light 1-4, fan 1-2)
- Natural variations of commands
- Multiple phrasings for same action

---

## 🔧 If Voice Not Working

1. **Check microphone permission** - App needs access
2. **Increase volume** - Make sure device is not muted
3. **Check internet** - Voice recognition needs connection
4. **Try again** - Sometimes recognition needs a second try
5. **Use numbers** - "Turn on light 1" always works
