### How to Add Screenshots

1. Take screenshots of your running system
2. Save them in this folder (`docs/screenshots/`)
3. Update README.md to reference them:

```markdown
![Dashboard](./docs/screenshots/dashboard.png)
![Android App](./docs/screenshots/android_app.png)
![Hardware](./docs/screenshots/hardware_setup.png)
```

### Screenshot Guidelines

- **Resolution**: At least 1920x1080 for desktop, 1080x1920 for mobile
- **Format**: PNG for UI screenshots, JPG for hardware photos
- **File Size**: Optimize images (< 500KB each)
- **Naming**: Use lowercase with underscores (e.g., `web_dashboard.png`)
- **Content**: Show actual functionality, not placeholder data

### Recommended Tools

- **Windows**: Snipping Tool, ShareX
- **Screenshot Editing**: Paint.NET, GIMP
- **Image Optimization**: TinyPNG, Squoosh
