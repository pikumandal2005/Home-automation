About The Project

Welcome to the Smart Home Automation System, an innovative project dedicated to bringing convenient, web-controlled lighting and fan management into the modern home. As a two-member team of electrical engineers from Gkciet, we combined our academic expertise to design and implement a practical and cost-effective home automation solution.

Technology Stack

Hardware

ESP8266, LEDs

Firmware

C++ (Arduino)

Frontend

HTML, TailwindCSS, JS

Communication

MQTT Protocol

How It Works

The system leverages the ESP8266 Wi-Fi module as the central intelligence. This versatile micro-controller hosts a custom-developed webpage, acting as the intuitive control interface you are currently using. The real-world control of your appliances is managed through simple electronic circuits, interfacing directly with the ESP8266.

Key Features

  • Remote Accessibility: Control your home environment from any device connected to your local network.
  • Seamless Integration: A user-friendly web interface eliminates the need for complex, proprietary apps.
  • Practical Engineering: A demonstration of efficient hardware-software co-design using widely available components.
  • Real-time Sync: Device status is synchronized across all connected clients using MQTT.
